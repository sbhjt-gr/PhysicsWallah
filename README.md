# Quizzy

A quiz app built with Kotlin and Jetpack Compose for Android.

If you want to try the demo app, download & install the APK from the [releases section](https://github.com/sbhjt-gr/Quizzy/releases/tag/v1.0.0).

# Setup

1. Clone this repo using `git clone https://github.com/sbhjt-gr/Quizzy.git`
2. Open Android Studio and click on "Open" then navigate to the cloned folder
3. Wait for Gradle to sync all the dependencies, this might take a few minutes
4. Connect your Android device with USB debugging enabled or start an emulator
5. Click the Run button or press Shift+F10 to build and install the app

You'll need JDK 17 or higher and Android Studio Hedgehog or newer.

# Tech Stack

The app is built with Kotlin and uses Jetpack Compose for the UI. Navigation is handled by Navigation Compose and network requests go through Retrofit with Gson for JSON parsing. State management uses ViewModel with Kotlin Coroutines for async operations. The minimum Android version supported is API 24 (Android 7.0).
